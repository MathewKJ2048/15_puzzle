# 15_puzzle
The 15 puzzle written in java.

## installation:

Download [Puzzle_15.java](https://github.com/MathewKJ2048/15_puzzle/blob/main/src/Puzzle_15.java) and compile it. Run main()

## commands:  
```8 - up```  
```4 - left```  
```6 - right```   
```5 - down```  
```q - quit```
